GitHub Pages Templates

A registry of ready-to-deploy GitHub Pages starters. Each one ships the current GitHub Actions Pages workflow and the right base-path setup, so it deploys correctly to a user or project site with no fiddling. Preview them live, then scaffold one into your repo.

How to use a template

  1. With Copilot: install the create-gh-pages-site skill and say what you want — "an Astro blog for octocat/blog". It pulls from this registry.
  2. With the generator: node scripts/new-site.mjs <template> --repo <owner/name> (or from anywhere with --registry jongio/gh-pages-templates).
  3. Push to main, then set Settings → Pages → Source → GitHub Actions.
